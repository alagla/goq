Qubic Game Of Life
PoC with Qupla library for Go

The following demo uses original Qubic Game Of Life Qupla code written by Eric Hop
as PoC for the reference implementation of Qupla. The Qupla code can be found here.
This demo is an example of GOQ Qupla library for Go.

It demponstrates implementation of Qubic oracle which handles interaction with the user/browser and serves as interface of the real world to Qubic.

The aim of this demo is experimenting with the architecture of Qubic supervisor and Qubic applications. It also tests platform independence of Qubic as well as GOQ compatibility with the Qubic specs.
