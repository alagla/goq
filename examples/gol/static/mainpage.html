Qubic Game Of Life
PoC with Qupla library for Go

The following demo uses original Qubic Game Of Life Qupla code written by Eric Hop
as PoC for the reference implementation of Qupla. It can be seen as "Hellow, World" of Qubic. The Qupla code can be found here.
The demo uses GOQ, Qupla library for Go.

It implements Qubic oracle which interacts with the user/browser and serves as a link between the real world and the Qubic. Each calculation of Game Of Life's next generation is done on the server by Qupla code. The browser and the oracle are just processing user input and sends output to the browser. GOL Oracle also maintains current GOL map for each session.

The purpose of this demo is to experiment with the architecture of Qubic supervisor and Qubic applications in general. It also is a test of platform independence of Qubic as well as GOQ compatibility with the Qubic specs.
